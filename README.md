# Spatial Attribute Integration into Graph Embeddings

This repository contains the implementation, experiment notebooks and result/evaluation outputs for the master thesis:

**Integration of Spatial Attributes into Graph Embeddings to Enhance Spatial Query Performance**

Author: Diego Henriquez  
University: Riga Technical University  
Study programme: Computer Systems  
Year: 2026

## Overview

The purpose of this repository is to provide the full experimental code and result material used in the thesis.
This research investigates if the integration of spatial attributes into graph embeddings improves spatial query performance, especially for nearby search and spatial neighbour retrieval.
